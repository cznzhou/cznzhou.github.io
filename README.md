# cznzhou.github.io

> 🌐 个人主页 · 开发者名片 · GitHub Pages 静态站点

这是我的个人主页源代码仓库，托管在 [GitHub Pages](https://pages.github.com) 上，展示了我的简介、联系方式与技术爱好。

---

## ✨ 特性

- 清新简约的毛玻璃卡片设计风格
- 响应式布局，完美适配移动端与桌面端
- 一键复制邮箱功能（Toast 提示）
- 动效柔和：卡片悬浮上移、按钮 hover 放大
- 纯原生 HTML + CSS + JavaScript，零依赖
